### 说明

`下载模板` -> `填写表单` -> `上传表单检查` -> `下载检查结果` -> `粘贴到表单` -> `完成`

</br>

### 操作步骤

**第1步** 下载表单模板，点击右侧下载按钮    

**第2步** 填写表单，只用填写表单中红色背景的区域    

如何选择P7_index? 请看这里 [Choose index1](http://192.168.206.171:6611/tools/hiseq_index)   

**第3步** 检查表单，通过右侧`上传`按钮，将填写好的表单上传    

**第4步** 检查结果，下表中 `Status`显示全是 OK，表示表单没问题。
